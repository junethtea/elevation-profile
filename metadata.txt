about=Elevation Profile is an essential tool for Network Planning and Optimization (NPO) Engineers and GIS professionals. It allows users to instantly generate highly accurate 2D elevation profiles directly from loaded DEM/Raster data. Key features include:
    * Interactive Line Drawing: Draw a path on the map and instantly view the elevation cross-section to verify Line of Sight (LoS).
    * Dynamic Terrain Smoothing: Adjust Sigma values on the fly to filter noise from raw DEM sources like AW3D30, DEMNAS, or SRTM.
    * QGIS 4 Ready: Fully refactored to support PyQt6 while maintaining seamless backward compatibility with QGIS 3.x.
    * Real-Time Summary: Automatically calculates total distance, minimum, maximum, and average elevation.
    * High-Res Export: Export elevation profiles to PNG or scalable SVG formats for professional technical reporting.
    Bridging the gap between GIS contour data and RF Planning workflows, this tool makes physical terrain analysis faster and more intuitive without needing heavy RF software.
